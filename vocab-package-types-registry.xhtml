要約

このドキュメントは、Package Document メタデータ内の dc:type 要素で使用するために IDPF で認められた型のセットを定義する。

このドキュメントは有益であり、EPUB® 仕様の主要なバージョンと紐付かない。

改訂方針

このドキュメントは、新しい識別子が EPUB コミュニティによって認められたとき常に変更されることがある。ここで定義された型は、削除されることはないが、廃止になるかもしれない。

追加の要求は、EPUB 3 Revision Tracker を介して行わなければならない。

出版物の型

dictionary
EPUB 出版物が、辞書を表現していることをを識別する。より詳細な情報は EPUB Dictionaries and Glossaries specification を参照されたい。
distributable-object [draft]
EPUB 出版物が、配布可能なオブジェクトを含んでいることを識別する。より詳細な情報は EPUB Distributable Objects specification を参照されたい。
edupub [draft]
EPUB 出版物が、EDUPUB Profile と一致していることを識別する。
index
EPUB 出版物がインデックスを表現することを識別する。より詳細な情報は EPUB Indexes specification を参照されたい。
preview [draft]
EPUB 出版物は作品のプレビューであることを識別する。より詳細な情報は EPUB Previews 仕様を参照されたい。
widget [draft]
EPUB 出版物がウィジェットであることを識別する。より詳細な情報は EPUB Widget Packaging and Integration を参照されたい。
