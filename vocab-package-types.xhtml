EPUB Registry of Publication Types

要約

このドキュメントは、Package Document メタデータ内の dc:type 要素で使用するために IDPF で認められた型のセットを定義する。

このドキュメントは有益であり、EPUB® 仕様の主要なバージョンと紐付かない。

改訂方針

このドキュメントは、新しい識別子が EPUB コミュニティによって認められたとき常に変更されることがある。ここで定義された型は、削除されることはないが、廃止になるかもしれない。

追加の要求は、EPUB 3 Revision Tracker を介して行わなければならない。

出版物の型

型は、この時点では定義されていない。
