要約

このドキュメントは、Package Document の collection 要素で使用するために IDPF が定義したすべての役割を掲載している。

役割は典型的に IDPF サブ仕様で定義され、その場合、正式な定義へ助言は役割名を提供されるが、このドキュメントは、特定の仕様書と紐付いていない汎用的な役割のための規範的な定義を含む。

このドキュメントは、EPUB® 仕様の主要なバージョンと紐付かない。

適合性宣言

この文書内のキーワード、must、must not、REQUIRED、SHALL、SHALL NOT、should、should not、RECOMMENDED、may、OPTIONAL は、[RFC2119] の記述に従って解釈される。

このドキュメントで定義されたすべての役割は規範である。

この仕様のすべての例は有益な参考情報である。

改訂方針

このドキュメントは、IDPF Working Group で定義された新しい役割によって常に変更されることがある。ここで定義された役割は、削除されることはないが、廃止になるかもしれない。

役割

役割は、この時点では定義されていない。

拡張

collection 要素のサードパーティの実装のリストは、http://www.idpf.org/epub/extensions/roles の拡張 wiki を参照されたい。
