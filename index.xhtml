EPUB 3 仕様書・関連文書

EPUB 3 Overview

EPUB 3 Overview は、他の EPUB 3 文書に記述され、アクセシビリティ、メタデータ、そして多言語サポートを包含している横断的なコンテンツを再検討している EPUB 3 の全体の導入として役立ちます。

翻訳底本：“EPUB 3 Overview” (IDPF Recommended Specification 11 October 2011)

EPUB Publications 3.0

EPUB Publications 3.0 は、出版物のメタデータ、出版物に含まれるリソース、表示されるリソースの順番を Package Document に定義する仕様です。主に .opf ファイル内の metadata、manifest、spine 要素に含まれるそれぞれ dc:、item、itemref 要素を扱います。

翻訳底本：“EPUB Publications 3.0” (IDPF Recommended Specification 11 October 2011)

EPUB Content Documents 3.0

EPUB Content Documents 3.0 は、EPUBR Publications のコンテクスト内で使用するための HTML5、SVG、CSS のプロファイルを定義します。

翻訳底本：“EPUB Content Documents 3.0” (IDPF Recommended Specification 11 October 2011)

EPUB Open Container Format (OCF) 3.0

EPUB Open Container Format (OCF) 3.0 は、単一ファイルのコンテナに1つ以上の EPUBR Publication を構成する関連リソースのセットをカプセル化するためのファイル形式と処理モデルを定義しています。

翻訳底本：“EPUB Open Container Format (OCF) 3.0” (IDPF Recommended Specification 11 October 2011)

EPUB Media Overlays 3.0

EPUB Media Overlays 3.0 は、EPUB Content Document と同期する音声表現用に、SMIL（同期マルチメディア統合言語）、Package Document、EPUBR Style Sheet、および EPUB Content Document の使用方法を定義しています。

翻訳底本：“EPUB Media Overlays 3.0” (IDPF Recommended Specification 11 October 2011)

EPUB Canonical Fragment Identifier (epubcfi) Specification

EPUB Canonical Fragment Identifier (epubcfi) は、フラグメント識別子を使って EPUB Publication の任意のコンテンツを参照する標準方式を定めています。

翻訳底本：“EPUB Canonical Fragment Identifier (epubcfi) Specification” (IDPF Recommended Specification 11 October 2011)

EPUB 3 Changes from EPUB 2.0.1

EPUB 3 Changes from EPUB 2.0.1、EPUB 3 は、XML と Web 標準に基いて、デジタル出版物の交換と配信形式である EPUB の第三の主要な版を定義している。本仕様書は、以前にリリースされた EPUB 2.0.1 から EPUB 3 での変更点について有益な方法で説明する。

翻訳底本：“EPUB 3 Changes from EPUB 2.0.1” (IDPF Recommended Specification 11 October 2011)

EPUB 3 Fixed-Layout Documents

EPUB 3 Fixed-Layout Documents は、EPUB 3 固定レイアウトを表現するためのメタデータ要素を定義する仕様です。また固定レイアウトの時の XHTML、SVG、画像のレンダリング寸法を表現する方法も定義しています。

翻訳底本：“EPUB 3 Fixed-Layout Documents” (IDPF Informational Document)

EPUB 3 Structural Semantics Vocabulary

EPUB 3 Structural Semantics Vocabulary は、出版物のコンテンツ内にセマンティック性を付与する epub:type 属性の値について定義しています。

翻訳底本：“EPUB 3 Structural Semantics Vocabulary” (IDPF)

W3C 仕様書 日本語訳

CSS Device Adaptation

この仕様は初期包含ブロック（initial containing block）に基いて使用されるビューポートのサイズや拡大率、向きを CSS で指定する製作者ための方法を提供する。

翻訳底本：“CSS Device Adaptation” (W3C Working Draft 15 September 2011)

HTML で WAI-ARIA を使う（Using WAI-ARIA in HTML）

このドキュメントは、障害を持つ人々にとって Web コンテンツと Web アプリケーションがよりアクセシブルになる方法を定義した Accessible Rich Internet Applications 仕様を用いて HTML 要素にアクセシビリティ情報を追加する方法を開発者に向けた実装ガイドです。

翻訳底本：“Using WAI-ARIA in HTML” (W3C Working Draft 03 October 2013)
