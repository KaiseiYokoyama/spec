EPUB 3.1 仕様書

EPUB 3.1 Overview

EPUB 3.1 Overview は、他の EPUB 3.1 文書に記述され、アクセシビリティ、メタデータ、そして多言語サポートを包含している横断的なコンテンツを再検討している EPUB 3.1 の全体の導入として役立つ。

修正日：2016年3月14日　 公開日：2016年3月8日

翻訳底本：“EPUB 3 Overview” (IDPF Editor's Draft 30 January 2016)

EPUB Packages 3.1

EPUB Packages 3.1 は、セマンティクスと EPUB® Package の適合必要条件を定める。各 Packages は EPUB 出版物の Rendition を表現し、Publication Resources に関連させる方法の要件のセットと Rendition のコンテンツを説明している Package Document によって定義される。

修正日：2016年3月14日　 公開日：2016年3月8日

翻訳底本：“EPUB Packages 3.1” (IDPF Editor's Draft 30 January 2016)

EPUB 3.0.1 仕様書・関連文書

EPUB 3 Overview

EPUB 3 Overview は、他の EPUB 3 文書に記述され、アクセシビリティ、メタデータ、そして多言語サポートを包含している横断的なコンテンツを再検討している EPUB 3 の全体の導入として役立ちます。

修正日：2014年6月26日　 公開日：2013年11月8日

翻訳底本：“EPUB 3 Overview” (IDPF Recommended Specification 26 June 2014)

EPUB Publications 3.0.1

EPUB Publications 3.0.1 は、出版物のメタデータ、出版物に含まれるリソース、表示されるリソースの順番を Package Document に定義する仕様です。主に .opf ファイル内の metadata、manifest、spine 要素に含まれるそれぞれ dc:、item、itemref 要素を扱います。

修正日：2014年6月26日　 公開日：2013年11月8日

翻訳底本：“EPUB Publications 3.0.1” (IDPF Recommended Specification 26 June 2014)

EPUB Content Documents 3.0.1

EPUB Content Documents 3.0.1 は、EPUBR Publications のコンテクスト内で使用するための HTML5、SVG、CSS のプロファイルを定義します。

修正日：2015年10月9日　 公開日：2013年11月8日

翻訳底本：“EPUB Content Documents 3.0.1” (IDPF Recommended Specification 26 June 2014)

EPUB Open Container Format (OCF) 3.0.1

EPUB Open Container Format (OCF) 3.0.1 は、単一ファイルのコンテナに1つ以上の EPUBR Publication を構成する関連リソースのセットをカプセル化するためのファイル形式と処理モデルを定義しています。

修正日：2014年6月26日　 公開日：2013年11月8日

翻訳底本：“EPUB Open Container Format (OCF) 3.0.1” (IDPF Recommended Specification 26 June 2014)

EPUB Media Overlays 3.0.1

EPUB Media Overlays 3.0.1 は、EPUB Content Document と同期する音声表現用に、SMIL（同期マルチメディア統合言語）、Package Document、EPUBR Style Sheet、および EPUB Content Document の使用方法を定義しています。

修正日：2014年6月26日　 公開日：2013年11月8日

翻訳底本：“EPUB Media Overlays 3.0.1” (IDPF Recommended Specification 26 June 2014)

EPUB Canonical Fragment Identifier (epubcfi) Specification

EPUB Canonical Fragment Identifier (epubcfi) は、フラグメント識別子を使って EPUB Publication の任意のコンテンツを参照する標準方式を定めています。

修正日：2014年6月26日　 公開日：2013年11月8日

翻訳底本：“EPUB Canonical Fragment Identifier (epubcfi) Specification” (IDPF Recommended Specification 26 June 2014)

EPUB 3.0.1 Changes from EPUB 3.0

EPUB 3.0.1 Changes from EPUB 3 は、重要な変更と追加を強調し、EPUB 3.0 仕様の最初の小改訂で行った変更を説明する。

修正日：2014年6月26日　 公開日：2013年11月8日

翻訳底本：“EPUB 3.0.1 Changes from EPUB 3.0” (IDPF Recommended Specification 26 June 2014)

EPUB 関連文書

EPUB 3 Structural Semantics Vocabulary

EPUB 3 Structural Semantics Vocabulary は、出版物のコンテンツ内にセマンティック性を付与する epub:type 属性の値について定義しています。

修正日：2015年9月28日　 公開日：2013年11月8日

翻訳底本：“EPUB 3 Structural Semantics Vocabulary” (IDPF)

EPUB Link Relationships Vocabulary

EPUB Link Relationships Vocabulary は、EPUB® Package Document の link 要素 rel 属性を利用するとき、リーディングシステムが認識するのに必要な関連する値のセットを定義する。

修正日：2014年3月31日　 公開日：2012年3月13日

翻訳底本：“EPUB Link Relationships Vocabulary” (IDPF)

EPUB Publications Reserved Prefixes

EPUB Publications Reserved Prefixes は、EPUB® Package Document のメタデータを使用するときに、リーディングシステムが認識するために必要な接頭辞のセットを定義する。

修正日：2015年7月7日　 公開日：2012年3月13日

翻訳底本：“EPUB Publications Reserved Prefixes” (IDPF)

EPUB Content Documents Reserved Prefixes

EPUB Content Documents Reserved Prefixes は、Content Document の EPUB® epub:type 属性を使用するときに、リーディングシステムが認識するために必要な接頭辞のセットを定義する。

修正日：2014年11月25日　 公開日：2012年3月13日

翻訳底本：“EPUB Content Documents Reserved Prefixes” (IDPF)

EPUB Registry of Collection Roles

EPUB Registry of Collection Roles は、Package Document の collection 要素で使用するために IDPF が定義したすべての役割を掲載している。

修正日：2015年7月7日　 公開日：2012年3月13日

翻訳底本：“EPUB Registry of Collection Roles” (IDPF)

EPUB Registry of Publication Types

EPUB Registry of Publication Types は、Package Document メタデータ内の dc:type 要素で使用するために IDPF で認められた型のセットを定義する。

修正日：2014年12月22日　 公開日：2012年3月13日

翻訳底本：“EPUB Registry of Publication Types” (IDPF)

Alternate Style Tags

このモジュールは、microformat を使用して代替スタイルシートのタグ付するためのパターンを定義している。

修正日：2014年6月26日　 公開日：2013年11月8日

翻訳底本：“Alternate Style Tags” (IDPF Recommended Specification 26 June 2014)

EPUB Indexes 1.0

EPUB Indexes 1.0 の目的は、すべての EPUB リーディングシステムでレンダリングされるインデックスを可能にする方法で、EPUB 出版物のインデックスの構造とコンテンツをエンコードの一貫性のある方法を定義し、仕様に準拠している EPUB リーディングシステム上で最適な方法で取り扱うことである。

修正日：2015年9月28日　 公開日：2014年2月17日

翻訳底本：“EPUB Indexes 1.0” (IDPF Recommended Specification 26 August 2015 )

EPUB Collection Element Role Extensions

EPUB Collection Element Role Extensions は、EPUB Package Document の collection 要素のために定義されたカスタムロールを識別する。

公開日：2015年1月30日

翻訳底本：“EPUB Collection Element Role Extensions” (IDPF)

EPUB 3 Magazine Navigation Vocabulary

EPUB 3 Magazine Navigation Vocabulary は、雑誌特有の構造的な語彙を表現している。

公開日：2014年11月25日

翻訳底本：“EPUB 3 Magazine Navigation Vocabulary” (IDPF)

EPUB Manifest Role

EPUB Manifest Role は、EPUB ® 出版物の collection [Publications301] の処理とレンダリングに必要なリソースのマニフェストを包含するための必要要件を定義する。

公開日：2015年1月30日

翻訳底本：“EPUB Manifest Role” (IDPF)

EPUB Previews 1.0

EPUB Previews は、立ち読みコンテンツを、EPUB® 出版物に包含することができる方法について説明している。

修正日：2015年9月30日　 公開日：2015年9月8日

翻訳底本：“EPUB Previews 1.0” (IDPF Recommended Specification 26 August 2015)

W3C 仕様書 日本語訳

CSS Device Adaptation

この仕様は初期包含ブロック（initial containing block）に基いて使用されるビューポートのサイズや拡大率、向きを CSS で指定する製作者ための方法を提供する。

公開日：2013年5月27日

翻訳底本：“CSS Device Adaptation” (W3C Working Draft 15 September 2011)

Using WAI-ARIA in HTML

このドキュメントは、障碍に接している人々にとって Web コンテンツと Web アプリケーションがよりアクセシブルになる方法を定義した Accessible Rich Internet Applications 仕様を用いて HTML 要素にアクセシビリティ情報を追加する方法を開発者に向けた実装ガイドです。

修正日：2015年3月9日　 公開日：2014年4月9日

翻訳底本：“Using WAI-ARIA in HTML” (W3C Working Draft 26 June 2014)

EPUB 3 Changes from EPUB 2.0.1
