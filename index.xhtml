EPUB 3.0.1 仕様書・関連文書

EPUB 3 Overview

EPUB 3 Overview は、他の EPUB 3 文書に記述され、アクセシビリティ、メタデータ、そして多言語サポートを包含している横断的なコンテンツを再検討している EPUB 3 の全体の導入として役立ちます。

改定日：2014年3月12日　 公開日：2013年11月8日

翻訳底本：“EPUB 3 Overview” (IDPF Proposed Specification 28 February 2014)

EPUB Publications 3.0.1

EPUB Publications 3.0.1 は、出版物のメタデータ、出版物に含まれるリソース、表示されるリソースの順番を Package Document に定義する仕様です。主に .opf ファイル内の metadata、manifest、spine 要素に含まれるそれぞれ dc:、item、itemref 要素を扱います。

改定日：2014年3月12日　 公開日：2013年11月8日

翻訳底本：“EPUB Publications 3.0.1” (IDPF Proposed Specification 28 February 2014)

EPUB Content Documents 3.0.1

EPUB Content Documents 3.0.1 は、EPUBR Publications のコンテクスト内で使用するための HTML5、SVG、CSS のプロファイルを定義します。

改定日：2014年3月12日　 公開日：2013年11月8日

翻訳底本：“EPUB Content Documents 3.0.1” (IDPF Proposed Specification 28 February 2014)

EPUB Open Container Format (OCF) 3.0.1

EPUB Open Container Format (OCF) 3.0.1 は、単一ファイルのコンテナに1つ以上の EPUBR Publication を構成する関連リソースのセットをカプセル化するためのファイル形式と処理モデルを定義しています。

改定日：2014年3月12日　 公開日：2013年11月8日

翻訳底本：“EPUB Open Container Format (OCF) 3.0.1” (IDPF Proposed Specification 28 February 2014)

EPUB Media Overlays 3.0.1

EPUB Media Overlays 3.0.1 は、EPUB Content Document と同期する音声表現用に、SMIL（同期マルチメディア統合言語）、Package Document、EPUBR Style Sheet、および EPUB Content Document の使用方法を定義しています。

改定日：2014年3月12日　 公開日：2013年11月8日

翻訳底本：“EPUB Media Overlays 3.0.1” (IDPF Proposed Specification 28 February 2014)

EPUB Canonical Fragment Identifier (epubcfi) Specification

EPUB Canonical Fragment Identifier (epubcfi) は、フラグメント識別子を使って EPUB Publication の任意のコンテンツを参照する標準方式を定めています。

改定日：2014年3月12日　 公開日：2013年11月8日

翻訳底本：“EPUB Canonical Fragment Identifier (epubcfi) Specification” (IDPF Proposed Specification 28 February 2014)

EPUB 3.0.1 Changes from EPUB 3.0

EPUB 3.0.1 Changes from EPUB 3 は、重要な変更と追加を強調し、EPUB 3.0 仕様の最初の小改訂で行った変更を説明する。

改定日：2014年3月12日　 公開日：2013年11月8日

翻訳底本：“EPUB 3.0.1 Changes from EPUB 3.0” (IDPF Proposed Specification 28 February 2014)

Alternate Style Tags

このモジュールは、microformat を使用して代替スタイルシートのタグ付するためのパターンを定義している。

改定日：2014年3月12日　 公開日：2013年11月8日

翻訳底本：“Alternate Style Tags” (IDPF Proposed Specification 28 February 2014)

EPUB Indexes 1.0

EPUB Indexes 1.0 の目的は、すべての EPUB リーディングシステムでレンダリングされるインデックスを可能にする方法で、EPUB 出版物のインデックスの構造とコンテンツをエンコードの一貫性のある方法を定義し、仕様に準拠している EPUB リーディングシステム上で最適な方法で取り扱うことである。

改定日：2014年2月28日　 公開日：2014年2月17日

翻訳底本：“EPUB Indexes 1.0” (IDPF Proposed Specification 28 February 2014)

EPUB 3 Structural Semantics Vocabulary

EPUB 3 Structural Semantics Vocabulary は、出版物のコンテンツ内にセマンティック性を付与する epub:type 属性の値について定義しています。

改定日：2014年3月12日　 公開日：2013年11月8日

翻訳底本：“EPUB 3 Structural Semantics Vocabulary” (IDPF)

EPUB 3 Changes from EPUB 2.0.1

EPUB 3 Changes from EPUB 2.0.1、EPUB 3 は、XML と Web 標準に基いて、デジタル出版物の交換と配信形式である EPUB の第三の主要な版を定義している。本仕様書は、以前にリリースされた EPUB 2.0.1 から EPUB 3 での変更点について有益な方法で説明する。

EPUB 関連文書

EPUB Link Relationships Vocabulary

EPUB Link Relationships Vocabulary は、EPUB® Package Document の link 要素 rel 属性を利用するとき、リーディングシステムが認識するのに必要な関連する値のセットを定義する。

改定日：2014年3月31日　 公開日：2012年3月13日

翻訳底本：“EPUB Link Relationships Vocabulary” (IDPF)

EPUB Publications Reserved Prefixes

EPUB Publications Reserved Prefixes は、EPUB® Package Document のメタデータを使用するときに、リーディングシステムが認識するために必要な接頭辞のセットを定義する。

公開日：2012年3月13日

翻訳底本：“EPUB Publications Reserved Prefixes” (IDPF)

コレクション ロール（Role）の EPUB レジストリ

「コレクション ロール（Role）の EPUB レジストリ」は、Package Document の collection 要素で使用するために IDPF が定義したすべての役割を掲載している。

改定日：2014年5月19日　 公開日：2012年3月13日

翻訳底本：“EPUB Registry of Collection Roles” (IDPF)

出版物タイプの EPUB レジストリ

EPUB Registry of Publication Types は、Package Document メタデータ内の dc:type 要素で使用するために IDPF で認められた型のセットを定義する。

改定日：2014年5月19日　 公開日：2012年3月13日

翻訳底本：“EPUB Registry of Publication Types” (IDPF)

EPUB Content Documents Reserved Prefixes

EPUB Content Documents Reserved Prefixes は、Content Document の EPUB® epub:type 属性を使用するときに、リーディングシステムが認識するために必要な接頭辞のセットを定義する。

公開日：2012年3月13日

翻訳底本：“EPUB Content Documents Reserved Prefixes” (IDPF)

W3C 仕様書 日本語訳

CSS Device Adaptation

この仕様は初期包含ブロック（initial containing block）に基いて使用されるビューポートのサイズや拡大率、向きを CSS で指定する製作者ための方法を提供する。

公開日：2013年5月27日

翻訳底本：“CSS Device Adaptation” (W3C Working Draft 15 September 2011)

HTML で WAI-ARIA を使う（Using WAI-ARIA in HTML）

このドキュメントは、障害を持つ人々にとって Web コンテンツと Web アプリケーションがよりアクセシブルになる方法を定義した Accessible Rich Internet Applications 仕様を用いて HTML 要素にアクセシビリティ情報を追加する方法を開発者に向けた実装ガイドです。

公開日：2014年4月9日

翻訳底本：“Using WAI-ARIA in HTML” (W3C Working Draft 03 October 2013)
