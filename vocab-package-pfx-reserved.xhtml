要約

このドキュメントは、EPUB® Package Document のメタデータを使用するときに、リーディングシステムが認識するために必要な接頭辞のセットを定義する。

宣言や Package Document でそれらを使用して方法など、接頭辞に関するより詳細な情報については、Publication 仕様内の語彙の関連付けメカニズムを参照されたい。

改訂方針

このドキュメントは、常に変更されることがある。ここで定義された接頭辞は、削除されることはないが、廃止になるかもしれない。

追加や変更、説明の要求は、EPUB 3 Revision Tracker を介して行わなければならない。
