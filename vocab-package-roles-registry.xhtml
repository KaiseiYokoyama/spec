要約

このドキュメントは、Package Document の collection 要素で使用するために IDPF が定義したすべての役割を掲載している。

このドキュメントは、EPUB® 仕様の主要なバージョンと紐付かない。

改訂方針

このドキュメントは、IDPF Working Group で定義された新しい役割によって常に変更されることがある。ここで定義された役割は、削除されることはないが、廃止になるかもしれない。

役割

dictionary
辞書の単一のインスタンスを構成する Content Document のセットを識別する。より詳細な情報は EPUB Dictionaries and Glossaries specification を参照されたい。
distributable-object [draft]
配布可能なオブジェクトの処理/レンダリングのために必要なリソースのセットを識別する。より詳細な情報は EPUB Manifest Role specification を参照されたい。
index
インデックスの単一のインスタンスを構成する Content Document のセットを識別する。より詳細な情報は EPUB Indexes specification を参照されたい。
manifest [draft]
コレクションの処理/レンダリングのために必要なリソースのセットを識別する。より詳細な情報は EPUB Manifest Role specification を参照されたい。
preview [draft]
EPUB 出版物のプレビューを生成するのに使用される Content Document のセットを識別する。より詳細な情報は EPUB Previews を参照されたい。
widget [draft]
埋め込まれた EPUB ウィジェットを構成するリソースのセットを識別する。より詳細な情報は EPUB Widget Packaging and Integration を参照されたい。

拡張

collection 要素のサードパーティの実装のリストは、http://www.idpf.org/epub/extensions/roles の拡張 wiki を参照されたい。
