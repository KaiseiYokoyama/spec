1 要約

1.1 この語彙について

このセクションは規定ではない。

この語彙は、パッケージ ドキュメントのメタデータに EPUB 出版物のアクセシビリティを記述するためのプロパティを定義する。

この語彙は、EPUB アクセシビリティ仕様の仲間であり、そのドキュメントと共に読まれることを意図している。

1.2 参照

この語彙を参照するためのベース IRI は、http://www.idpf.org/epub/vocab/package/a11y/# である。

接頭辞「a11y:」は、この語彙のプロパティで使用するために予約済みであり、パッケージ ドキュメントで宣言する必要はない。

1.3 用語

EPUB 固有の意味を持つ用語は、（例えば、「製作者（Author）」「リーディング システム（Reading System）」など）このドキュメントでは大文字にしている。それらの意味については、EPUB Accessibility 仕様を見よ。

セクションにある用語の最初のインスタンスのみが、その定義にリンクされる。

2 適合性プロパティ

2.1 certifiedBy

名前： certifiedBy
概要： EPUB 出版物のアクセシビリティの検査と証明を担当する当事者を識別する。
許可された値： xsd:string
個数： ひとつ以上
例： <meta property="a11y:certifiedBy">Accessibility Testers Group</meta>

2.2 certifierCredential

名前： certifierCredential
概要： アクセシブルなコンテンツを認定するために、certifiedBy プロパティで識別された当事者の権限を確立する資格や印を識別する。
許可された値： xsd:string
個数： 0 個以上
例： <meta property="a11y:certifierCredential">DAISY OK</meta>

2.3 certifierReport

名前： certifierReport
概要： certifiedBy プロパティで識別される当事者によって作成されたアクセシビリティ レポートへのリンクを提供する。
許可された値： xsd:anyURI
個数： 0 個以上
例： <link rel="a11y:certifierReport" href="http://example.com/a11y/reports/9780000000001"/>
