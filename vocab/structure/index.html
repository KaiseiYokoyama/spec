このドキュメントのステータス

このドキュメントは、EDUPUB profile からのドラフトの用語が含まれている。

Indexes セクションのすべての用語は、index 用語を除いて、現在、Proposed Specification である EPUB Indexes 仕様の新しい機能である。

[draft] マークの用語の使用はいずれも、実験的であると考慮するべきである。

廃止予定の用語は、[deprecated] によって識別できる。これらの用語は、もはや使用することは推奨されない。

追加、変更、および明確化のリクエストは、イシュートラッカーを通じて行うことができる。プルリクエストも歓迎している。

変更リクエストのプロセスの詳細については、IDPF Issue Reporting Guidelines を参照されたい。

語彙について

この語彙（EPUB® 3 の一部）は、書かれた作品の構造意味論の記述に関連するプロパティのセットを定義する。語彙は典型的にホスト言語に依存せず、主に HTML 語彙の要素の意味的な抑揚を有効にするために構成されている。

HTML で使用するコンテクストフィールドは、指定されたプロパティが適切と見なされている HTML ドキュメントのコンテクストを示している。製作者は、特に記載されていない HTML のマークアップ要素にプロパティを利用してもよいが（may）、セマンティクスは、それらが含んでいる要素のセマンティクスのサブセットを表現することを示し、意味論的に中性な要素に既存の要素の意味を加えないことを確保しなければならない（must）。

HTML ドキュメントを処理するとき、使用コンテクストがホストの指定によって明示的に上書きまたは拡張でない限り、リーディングシステムはこのような非適合のプロパティを無視してもよい。

使用方法の詳細セクションでは、指定されたプロパティを定義または利用している IDPF 仕様を特定する。

ドキュメントのパーティション

cover

マーケティング用の画像、タイトル、著者や出版者、および厳選した引用とレビューで構成されている作品を紹介するセクション。

HTML で使用可能な場所：section 要素、body 要素

frontmatter

出版物の主要なコンテンツの予備的な資料、例えば目次や献辞など。

HTML で使用可能な場所：section 要素、body 要素

bodymatter

出版物のメインコンテンツ。

HTML で使用可能な場所：section 要素、body 要素

backmatter

出版物の主要なコンテンツの後に在る補助的な資料、例えば索引や付録など。

HTML で使用可能な場所：section 要素、body 要素

特定の主題、物語の一部分または同様のカプセル化されたテーマを扱う関連したセクションのセットを含む作品の主要な構造ディビジョン。

HTML で使用可能な場所：section 要素、body 要素

HTML で使用可能な場所：section 要素、body 要素

HTML で使用可能な場所：section 要素、body 要素

ドキュメントのセクションとコンポーネント

典型的に出版物の bodymatter に生じるセクションとコンポーネント。

abstract [draft]

作品またはセクションの本質的なアイデア、コンセプトと結び、またはその中に引用する短い概要。

HTML で使用可能な場所：section 要素, グルーピング コンテンツ

foreword

作品の前の導入部分。典型的に作品の著者によって書かれない。

HTML で使用可能な場所：section 要素、グルーピング コンテンツ

preface

作品の前の導入部分。典型的に作品の作者によって書かれる。

HTML で使用可能な場所：section 要素、グルーピング コンテンツ

prologue

作品の背景を設定する導入部分。典型的に物語そのもの。

HTML で使用可能な場所：section 要素、グルーピング コンテンツ

introduction

作品の範囲または性質を紹介する典型的に予備的なセクション。

HTML で使用可能な場所：section 要素、グルーピング コンテンツ

preamble

作品の始まりのセクション。典型的に作品の内容の範囲または性質に関して紹介および／または説明的な散文を含む。

HTML で使用可能な場所：section 要素、グルーピング コンテンツ

conclusion

作品を要約したり、物語をまとめた結論のセクションまたは声明。

HTML で使用可能な場所：section 要素、グルーピング コンテンツ

epilogue

典型的には将来の視点から、作品の活動や出来事をまとめたりコメントしたりする物語のまとめのセクション。

HTML で使用可能な場所：section 要素、グルーピング コンテンツ

afterword

著者または重要な人物による締めくくりの言葉で、典型的に、コンテンツが書かれたことについての洞察や、そのタイムラインから発生した重要または関連するイベントを提供する。

HTML で使用可能な場所：section 要素、グルーピング コンテンツ

epigraph

作品の開始時に設定された引用句、またはテーマを設置したり、雰囲気を設定するセクション。

HTML で使用可能な場所：section 要素、グルーピング コンテンツ

使用方法の詳細：EPUB Indexes – index-headnotes プロパティ

index-legend

記号、略語または index で使用される特殊な書式とその意味の一覧。

必須の親コンテクスト： index-headnotes

HTML で使用可能な場所：セクショニング コンテンツ、dl 要素

使用方法の詳細：EPUB Indexes – index-legend プロパティ

index-group

例えば、メインエントリの先頭文字（starting letter）など、共通の特徴を共有する連続したメインエントリのコレクション。

必須の親コンテクスト： index

HTML で使用可能な場所：セクショニング コンテンツ

使用方法の詳細：EPUB Indexes – index-group プロパティ

index-entry-list

連続したメインエントリまたはサブエントリのコレクション。

必須の親コンテクスト： index-entry、 index-group と index

HTML で使用可能な場所：ul 要素。このプロパティは、index-headnotes 内を除いて、ul が index の先祖を持つ場合、暗黙的に指定される

使用方法の詳細：EPUB Indexes – index-entry-list プロパティ

index-entry

任意の付帯するサブエントリ、ロケータ、相互参照と/または編集者注を持つある用語。

必須の親コンテクスト：index-entry-list

HTML で使用可能な場所：li 要素。このプロパティは、親 ul が型 index-entry-list（暗黙的または明示的）の場合、暗黙的に指定される

使用方法の詳細：EPUB Indexes – index-entry プロパティ

index-term

単語、語句、文字列、グリフまたはインデックス可能なコンテンツを表す画像。

必須の親コンテクスト：index-xref-related、 index-entry と index-xref-preferred

HTML で使用可能な場所：フレージング コンテンツ；典型的に span 要素

使用方法の詳細：EPUB Indexes – index-term プロパティ

index-editor-note

単一のエントリに関連する編集者注。

必須の親コンテクスト： index-entry

HTML で使用可能な場所：フロー コンテンツ

使用方法の詳細：EPUB Indexes – index-editor-note プロパティ

index-locator

出版物内のインデックス化されたコンテンツの存在を参照。

必須の親コンテクスト：index-entry、index-locator-list と index-locator-range

HTML で使用可能な場所：a 要素。このプロパティは、親コンテクストが、index-locator-list または index-locator-range の場合、暗黙的に指定される

使用方法の詳細：EPUB Indexes – index-locator プロパティ

index-locator-list

連続ロケータまたはロケータの範囲のコレクション。

必須の親コンテクスト：index-entry

HTML で使用可能な場所：ul 要素

使用方法の詳細：EPUB Indexes – index-locator-list プロパティ

index-locator-range

単一のポイントよりはむしろコンテンツの範囲に用語を接続するロケータの組み合わせ。

必須の親コンテクスト：index-locator-list と index-entry

HTML で使用可能な場所：フロー コンテンツ

使用方法の詳細：EPUB Indexes – index-locator-range プロパティ

index-xref-preferred

インデックス内のひとつの用語からひとつ以上の推奨する用語または用語カテゴリの参照（“XXXを見よ”に類似）。

必須の親コンテクスト：index-entry

HTML で使用可能な場所：フロー コンテンツ

使用方法の詳細：EPUB Indexes – index-xref-preferred プロパティ

index-xref-related

インデックス内のひとつの用語からひとつ以上の関連する用語または用語カテゴリの参照（“XXXも見よ”に類似）。

必須の親コンテクスト：index-entry

HTML で使用可能な場所：フロー コンテンツ

使用方法の詳細：EPUB Indexes – index-xref-related プロパティ

index-term-category

単語、語句、文字列、グリフまたはインデックス内の用語のカテゴリを表す画像。

必須の親コンテクスト：index-xref-related と index-xref-preferred

HTML で使用可能な場所：a 要素

使用方法の詳細：EPUB Indexes – index-term-category プロパティ

index-term-categories

インデックスに関連している用語カテゴリのリストのためのラッパー。

必須の親コンテクスト：index-xref-related と index-xref-preferred

HTML で使用可能な場所：a 要素

使用方法の詳細：EPUB Indexes – index-term-categories プロパティ

HTML で使用可能な場所：glossdef プロパティは、glossary プロパティを持つ dl 要素内の dd 要素へ、暗黙的に指定される。

タイトルと見出し

halftitle

作品の最初のページまたはテキストの直前に在るタイトル。

title も参照されたい。

HTML で使用可能な場所：ヘディング コンテンツ。このプロパティは、ドキュメントのスコープ内に一度だけ出現されるべきである。

fulltitle

作品のフルタイトル。title と同一または title と subtitle から成る複合語。

title も参照されたい。

http://purl.org/dc/terms/title と同様である。

HTML で使用可能な場所：ヘディング コンテンツ。このプロパティは、ドキュメントのスコープ内に一度だけ出現されるべきである。

covertitle

作品の表紙に表示されるような作品のタイトル。

title も参照されたい。

HTML で使用可能な場所：ヘディング コンテンツ。このプロパティは、ドキュメントのスコープ内に一度だけ出現されるべきである。

title

リスト、表または図形など、ドキュメントにあるコンポーネントの主要な名前。

HTML で使用可能な場所：ヘディング コンテンツ、ヘディング コンテンツの子孫にあたるフレージング コンテンツ。

subtitle

作品、またはその中にあるセクションもしくはコンポーネントのための説明または代替タイトル。

title も参照されたい。

HTML で使用可能な場所：ヘディング コンテンツ。ヘディング コンテンツ、p 要素、div 要素の子孫にあたるフレージング コンテンツ。

ordinal [draft]

コンポーネントの連続（例えば「1」、「IV」、「B-1」）内のコンポーネントの序列を指定する。
